# Google Business Profile Actions

## Highest-impact actions

- Set primary category as close as possible to the main business.
- Add supporting categories if accurate.
- Make sure name, phone, website, and hours exactly match the site.
- Add service descriptions that match the website language.
- Upload real photos regularly.
- Publish one GBP update per week.
- Ask for reviews every week.
- Reply to every review.

## Suggested weekly GBP post themes

- hiring in Greensboro
- hiring in High Point
- hiring in Winston-Salem
- sales internship for students
- entry-level sales opportunity
- young professionals building income and communication skill

## Review collection focus

- ask happy applicants for feedback on communication and professionalism
- ask partners/customers for trust and responsiveness feedback
- never fake or bulk-buy reviews

## What to track

- website clicks
- direction requests
- calls
- branded search volume
- review count
- review velocity
